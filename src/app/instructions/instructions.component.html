How do we build this app, what commands do we run?

  1. ng new pet-app
  2. cd pet-app
  3. ng serve -o
  4. ng generate component instructions
  5. ng generate component pet
  6. ng generate component pets
  7. ng generate component add-pet
  8. ng generate service services/pet
  9. ng generate component login
  10. ng generate component register
  11. ng generate service services/person
  12. ng generate service services/session
  13. npm install ngx-cookie
  14. Add the Cookie module to the imports in app.module.ts
  15. "ng test"to test our app
  16. "ng test --no-watch --code-coverage" to test with coverage
